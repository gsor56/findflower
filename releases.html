Release notes

What’s new in FindFlower.

FindFlower is an actively evolving product. Every release brings better identification, a calmer interface, and more of the natural world within reach. Here’s the trail so far.

Beta 0.9.4

Current 8 July 2026

New features

  • Identify flowers from a link, including automatic YouTube thumbnail extraction.
  • Developer preview: on-device SDK documentation and gated API keys via secure sign-in.

Model improvements

  • Expanded coverage to 107 botanical classes.
  • Steadier confidence scoring, with clearer separation between close look-alikes.

Interface

  • Refined result panel with animated confidence and top alternatives.
  • Live Wikipedia context, with reference image and full-article link.

Fixes

  • Resolved model-loading issues on browsers with strict cache settings.
  • Corrected preview handling for large photos.

Beta 0.9.0

21 June 2026

New features

  • Live camera capture, alongside image upload.
  • Feedback prompt after each result to guide future improvements.

Model improvements

  • Fully in-browser inference — images no longer leave the device.

Fixes

  • Improved camera permission handling across browsers.

Beta 0.8.0

2 June 2026

The first public Beta

  • Core identification from an uploaded photo.
  • The calm, single-workspace interface FindFlower is built around.
  • Initial model covering common garden flowers.

On the horizon

  • A hosted REST API, moving from preview to general availability.
  • Broader species coverage, informed by your feedback.
  • Saved history for signed-in users.
